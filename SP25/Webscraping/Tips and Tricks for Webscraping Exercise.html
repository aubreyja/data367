Common Issues (and some solutions):

  • "NA created by coercion" error: This means that you tried to convert a variable to a numeric that R wasn't able to fully convert - check that the column doesn't have any strange formatting (commas, colons, etc.)
  • CSS tag starts with .left: I'm still not 100% sure what causes this to happen, but just replace ".left" with "td". When in doubt, try to format things in ways that have worked in the past!

Useful R commands:

  • strsplit(string, sep): Takes a string and splits it into separate strings everywhere it finds the character given in "sep"
  • str_replace_all(string, pattern, replacement): Replaces all occurrences of 'pattern' in string with 'replacement'
  • trimws(string, which = 'both'): Removes all whitespace from both ends of the string.
  • as.numeric(gsub(",", "", y)) Removes commas from numbers in the vector y and reads the result as numbers

Note: To use some of these, you need to install the "stringr" package using the command: install.packages("stringr")

Variable Types:

You may have noticed that in the .rmd file I shared, I use things such as as.factor() and as.numeric(). When should you use those functions?

  • as.factor(): You have a categorical variable where there isn't any inherent order (such as conference)
  • as.numeric(): You have a quantitative variable (such as points or score)
